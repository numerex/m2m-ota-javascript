===================

Numerex Machine-to-Machine Over the Air Protocol for Javascript
-------------------

Currently contains support for the following object types:

* byte
* ubyte
* int
* uint
* string
* float (4-byte)
* double (8-byte)
* timestamp
* byte array
* ubyte array
* int array
* uint array
* float (4-byte) array
* double (8-byte) array

NOTE: only 2-byte integers are currently supported
